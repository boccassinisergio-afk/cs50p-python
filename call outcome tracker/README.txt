# Call Outcome Tracker
Record the outcome of every incoming call in relation to sales made and issues reported by customers.

## Features:
- Enter sales made for each customer served. 
- Enter any issues encountered by each customer (you can enter more than one per call). 
- The software stores this information in a database which can be accessed at the end of the day or on subsequent days.

## Usage:
- The menu is interactive and intuitive; press 1 to record a sale, 2 to enter one or more issues (or press ‘Exit’ if you wish to return to the main menu), 3 to view the daily report, 4 to exit.

## Technologies 
- Python and JSON.
